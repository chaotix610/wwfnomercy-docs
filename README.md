# wwfnomercy-docs

## WWF No Mercy Preservation Project

A comprehensive, community‑driven effort to **preserve, document, and understand every aspect of** _**WWF No Mercy**_ — the legendary 2000 Nintendo 64 wrestling game developed by Asmik Ace and AKI Corporation.

This repository serves as the technical, historical, and archival companion to the long‑running fan wiki at https://wwfnomercy.fandom.com, expanding the project into a structured, version‑controlled home for research, documentation, and preservation.

## Project Goals

- **Full technical documentation** of game systems, mechanics, data structures, and engine behavior
    
- **Preservation of assets** (within legal limits): textures, models, audio references, scripts, and configuration formats
    
- **Reverse‑engineering insights** into how the AKI engine works under the hood
    
- **Historical archiving** of development context, unused content, regional differences, and community discoveries
    
- **Modding resources** to support ROM hackers, texture modders, and tool developers
    
- **Long‑term stewardship** of knowledge that has been scattered across forums, dead links, and personal notes for decades
    

## What This Repo Contains

This repository is organized to make both casual browsing and deep research easy:

- **/docs/** — Detailed documentation of mechanics, systems, and internal structures
    
- **/research/** — Reverse‑engineering notes, findings, and technical breakdowns
    
- **/assets/** — Legally shareable reference materials (no copyrighted ROM data)
    
- **/tools/** — Scripts, utilities, and community‑made tools for analysis and modding
    
- **/history/** — Development history, unused content, version differences, and archival notes
    
- **/community/** — Links, credits, and documentation of community discoveries
    

Each section is designed to grow over time as more information is verified and added.

## Scope of Documentation

The long‑term goal is to fully document:

- **Gameplay systems** — grapples, counters, AI logic, damage, momentum, etc.
    
- **Match types & rulesets** — including hidden or unused variations
    
- **Character data** — stats, movesets, animations, and internal IDs
    
- **Move logic** — timing windows, priority, collision behavior
    
- **Engine internals** — memory maps, file formats, compression, scripting
    
- **Menus & UI** — structure, behavior, and quirks
    
- **Save data** — structure, corruption causes, and restoration research
    
- **Regional differences** — USA, EU, JPN, Rev A, and kiosk builds
    
- **Unused content** — moves, wrestlers, arenas, audio, and debug remnants
    

This project aims to be the **definitive technical reference** for the game.
